Tradução Tupi-Português
</h1>
<h2>Projeto</h2>

Projeto desenvolvido para Trabalho de Conclusão de Curso - TCC. A Api faz o serviço de tradução de uma frase ou palavra do Tupi Potiguara para o/a Português/Glosa.
